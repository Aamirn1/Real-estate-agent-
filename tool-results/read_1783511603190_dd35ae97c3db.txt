     1→# LeadSphere AI — Worklog & Design System
     2→
     3→## Project Overview
     4→Building "LeadSphere AI" — a premium AI-powered Real Estate Lead Generation SaaS landing page. Single route `/` in `src/app/page.tsx`. Dark luxury SaaS aesthetic.
     5→
     6→## Design System (ALL AGENTS MUST FOLLOW)
     7→
     8→### Theme
     9→- Deep black base `#050505`, dark graphite `#0d0d10`
    10→- Accent palette (use as Tailwind arbitrary values or theme colors):
    11→  - Electric blue `#3b82f6` → `bg-electric`, `text-electric`
    12→  - Violet `#8b5cf6` → `bg-violet`, `text-violet`
    13→  - Cyan `#06b6d4` → `bg-cyan`, `text-cyan`
    14→  - Gold `#d4af37` → `text-gold` (premium badges ONLY)
    15→- White text `#f5f5f7`, muted `text-white/55`, `text-white/70`
    16→
    17→### Typography
    18→- Headings: `font-heading` class (Space Grotesk) — use `font-heading` on h1-h6
    19→- Body: default `font-body` (Inter)
    20→- Numbers: add `tnum` class for tabular nums
    21→
    22→### Shared primitives (IMPORT from these, do NOT recreate)
    23→```tsx
    24→import { GlassCard, CountUp, SectionHeading, SectionShell } from "@/components/leadsphere/primitives";
    25→import { Reveal } from "@/components/leadsphere/Reveal";
    26→import { TiltCard } from "@/components/leadsphere/TiltCard";
    27→import { MagneticButton } from "@/components/leadsphere/MagneticButton";
    28→import { AnimatedBackground } from "@/components/leadsphere/AnimatedBackground";
    29→```
    30→- `GlassCard` props: `{ className?, strong?, sheen?, children }`
    31→- `CountUp` props: `{ value, duration?, decimals?, prefix?, suffix?, className? }`
    32→- `SectionHeading` props: `{ eyebrow?, title, description?, align?: 'left'|'center', className? }`
    33→- `SectionShell` props: `{ children, className?, id? }` — use to wrap each section
    34→- `Reveal` props: `{ children, className?, delay?, y? }` — scroll reveal wrapper
    35→
    36→### Glassmorphism utility classes (in globals.css)
    37→- `.glass` — subtle glass
    38→- `.glass-strong` — darker strong glass
    39→- `.glass-card` — gradient glass card
    40→- `.glass-sheen` — add reflection sheen (use as wrapper)
    41→
    42→### Animation classes (in globals.css)
    43→- `.animate-aurora`, `.animate-float-y`, `.animate-float-y-slow`, `.animate-pulse-ring`
    44→- `.animate-shimmer`, `.animate-gradient-x`, `.animate-spin-slow`, `.animate-marquee`, `.animate-orb-breathe`
    45→- `.bg-grid`, `.bg-grid-fade`, `.noise`
    46→
    47→### Gradient text helpers
    48→- `.text-gradient` — white→indigo→cyan
    49→- `.text-gradient-electric` — blue→violet→cyan
    50→
    51→### Glow helpers
    52→- `.glow-electric`, `.glow-violet`, `.glow-soft`, `.text-glow`
    53→
    54→### Existing shadcn/ui components available
    55→All in `src/components/ui/`. Common: button, card, tabs, accordion, badge, avatar, input, textarea, progress, slider, tooltip, dialog, sheet, carousel, etc. Prefer these over custom.
    56→
    57→### Tech
    58→- Next.js 16 App Router, TypeScript, Tailwind CSS 4, framer-motion (motion), lucide-react icons, recharts for charts
    59→- All section components MUST start with `"use client";`
    60→- Mobile-first responsive. Use `sm:`, `md:`, `lg:` breakpoints
    61→- Use framer-motion `whileInView` with `viewport={{ once: true }}` for scroll animations
    62→
    63→### File naming
    64→Each section lives in `src/components/leadsphere/sections/<SectionName>.tsx` and exports a default component.
